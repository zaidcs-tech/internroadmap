The 3 Golden BidaTech Rules

1. Branch Isolation

Never commit to main directly. Use Pull Requests for every feature.

2. Zero Leak Policy

.env and .db files must be in .gitignore. Revoke leaked keys immediately.

3. Defensive Coding

Always wrap AI engine blocks in try-except structures.
